3D Shoe Customizer using three.js for Swear.

Summary

Project Brief

For this school project in collaboration with Swear, we were challenged to bring a digital shoe to life by making it fully customizable in 3D. The goal was to allow users to interact with the product in a personal, playful and intuitive way.

Solution

I developed an interactive 3D shoe experience using Three.js, where users can explore the shoe model in real time and instantly change colours, textures and materials. I also designed a custom background in Figma, keeping the look and feel aligned with Swearâ€™s bold and playful vibe. It became a fully personalized, immersive experience. This project challenged me to bring together technical 3D rendering and UX thinking. I learned how to work with object hierarchies, map user events in 3D space, and keep performance smooth even with complex models. Tools I used: Three.js, JavaScript and Figma.

Key Features

  • Clickable 3D elements:

    • Each shoe part is individually clickable, allowing users to change materials(textures) and colors with smooth visual feedback.

  • Initials customization:

    • Users can enter their own initials, which are directly visualized on the shoe model.

  • Real-time rendering:

    • Seamless interactivity and dynamic updates as to provide an immersive and intuitive customised experience.
